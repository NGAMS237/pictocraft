Fichiers d'impression

Envoyez vos fichiers au format PDF, résolution 300 dpi, mode CMJN, avec 3mm de fonds perdus. Pas de fichier prêt ? Notre équipe d'infographistes peut le créer pour vous.

Procédure de commande

1. Configurez votre devis
2. Envoyez vos fichiers (ou commandez la création)
3. Validez le bon à tirer
4. Production lancée
5. Livraison Yaoundé/Douala

Délais de production

Standard : 3-5 jours ouvrables
Express : 24-48h (Yaoundé)
Urgent : jour même si commande avant 12h

Fichiers reçus avant 14h ? Production lancée le jour même.
