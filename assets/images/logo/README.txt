POUR REMPLACER PAR VOTRE VRAI LOGO :

1. Préparez votre logo officiel au format PNG ou SVG :
   - Largeur minimum : 600 px
   - Fond transparent recommandé
   - Format paysage (plus large que haut)

2. Renommez votre fichier en :
       logo.png       (si PNG)
       logo.svg       (si SVG)

3. Glissez-déposez ce fichier dans ce dossier :
       assets/images/logo/

4. Le logo apparaîtra automatiquement sur toutes les pages
   du site (header, footer, mobile menu).

LOGO PAR DÉFAUT :

Si aucun fichier "logo.png" ou "logo.svg" n'est trouvé,
le site affiche une version texte stylée du logo
(Pictocraft SARL) avec les couleurs officielles.
