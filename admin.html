Comment utiliser ce back-office ?

📌 Modifier les informations

Naviguez via le menu de gauche pour atteindre la section voulue : Entreprise, Contacts, Produits, Tarifs. Modifiez les champs directement dans les formulaires.

💾 Sauvegarder

Une fois vos modifications faites, allez dans Export / Import et cliquez sur "Télécharger config.js". Remplacez l'ancien fichier assets/js/config.js par celui que vous venez de télécharger.

🖼️ Changer une image produit

Préparez une nouvelle image au format JPG ou PNG (au moins 800×600px). Renommez-la exactement comme le fichier actuel (par exemple flyers.jpg). Glissez-la dans le dossier assets/images/products/ en remplaçant l'ancienne. Pas besoin de toucher au code.

🎨 Changer le logo

Préparez votre logo au format PNG (fond transparent, largeur 600px minimum) ou SVG. Nommez-le logo.png ou logo.svg et placez-le dans assets/images/logo/.

⚠️ Sécurité

Le fichier admin.html ne doit jamais être mis en ligne publiquement. Utilisez-le uniquement sur votre ordinateur. Sur le serveur web, supprimez ce fichier ou protégez-le par mot de passe.

🆘 Besoin d'aide ?

Contactez votre développeur web ou consultez le fichier GUIDE-MODIFICATION.md à la racine du projet.
