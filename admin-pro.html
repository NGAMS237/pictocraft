Méthode recommandée : gérer les images directement dans le repo GitHub
Ajoute, supprime ou renomme tes images via l'interface web GitHub. C'est plus fiable que l'upload et tout est sauvegardé automatiquement avec historique.
📸 Images produits 🎯 Images services 🖼️ Images réalisations ⬆ Ajouter directement
Comment faire ?
  1. Clique sur "📸 Images produits" pour voir tous les fichiers
  2. Pour ajouter : bouton Add file → Upload files → glisse tes images
  3. Pour supprimer : clique sur un fichier → icône poubelle 🗑️ en haut à droite
  4. Pour renommer : clique sur un fichier → icône crayon ✏️
  5. Après chaque modif : bouton Commit changes tout en bas
  6. Attends 1 minute — GitHub Pages rebuilde automatiquement
  7. La Galerie rapide dans l'onglet Produits verra tes nouvelles images
💾
Méthode alternative : Supabase Storage (upload direct depuis le navigateur, plus rapide mais URL longue). Utilise-la si tu veux juste tester rapidement.
